# Data Layout

- `raw/`: immutable source files collected directly from primary sources
- `external/`: third-party inputs or manually acquired reference files
- `processed/`: analysis-ready datasets generated by pipeline scripts
- `diagnostics/`: validation, matching, and QA outputs

This public-safe repository excludes raw corpus texts, full article text, and
proprietary export artifacts. Keep large or sensitive files out of Git unless
they are intentionally cleared for redistribution.
