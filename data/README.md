# Data Layout

- `raw/`: reserved for source files collected from primary sources; raw files are not included in this public repository.
- `external/`: reserved for third-party inputs or manually acquired reference files that cannot be redistributed here.
- `processed/`: public-safe, analysis-ready datasets derived for the thesis workflow.
- `diagnostics/`: compact validation, matching, and quality-assurance outputs.

The published repository excludes raw corpus texts, full article text,
proprietary exports, and other source materials that are not cleared for public
redistribution. The included files are intended to document the structure and
derived outputs of the analysis without republishing restricted inputs.
